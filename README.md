# Brafront - Backend Python

API REST simples para cadastro de clientes, produtos e pedidos.

## Tecnologias
- Python
- Flask
- SQLite
- Flask-CORS
- Werkzeug (hash de senha)

## Como executar
```bash
cd backend_python
python main.py
```

A API será iniciada em `http://localhost:5000`.
